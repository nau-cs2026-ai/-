# 🚀 快速同步代码到 GitHub

## 一键同步（推荐）

### 步骤 1：安装 Git
下载并安装 Git：https://git-scm.com/download/win

### 步骤 2：双击运行
双击 `sync-to-github.bat` 文件，脚本会自动：
- ✓ 检查 Git 安装
- ✓ 初始化仓库
- ✓ 配置远程地址
- ✓ 添加所有修改
- ✓ 提交代码
- ✓ 推送到 GitHub

### 步骤 3：输入仓库地址
首次运行时，脚本会提示输入 GitHub 仓库地址，例如：
```
https://github.com/你的用户名/仓库名.git
```
